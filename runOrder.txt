Here's the complete command sequence to run your analysis from start to finish:

## **Setup and Testing (Run Once)**

```bash
# 1. Fix Python imports (required first step)
python import_fix.py

# 2. Test basic functionality
python scripts/engage.py --target-x 30 --target-y 0 --target-z 15 --drone-size small
```

## **Core Analysis Generation (Main Results)**

```bash
# 3. Generate comprehensive single drone analysis
python examples/single_drone.py
# Output: results/single_drone_analysis.txt

# 4. Generate multi-target engagement analysis  
python examples/multiple_targets.py
# Output: results/multiple_targets_analysis.txt

# 5. Generate parametric optimization study
python examples/parametric_study.py
# Output: results/parametric_analysis.txt
```

## **Paper-Specific Validation Tests**

```bash

python scripts/engage.py --envelope-analysis --drone-type small
```

## **Execution Notes**

- **Commands 1-2**: Essential setup - must run first
- **Commands 3-5**: Generate your main analysis datasets for the paper
- **Commands 6-9**: Validate specific paper claims
- **Commands 10-12**: Create figures for paper/README
- **Commands 13**: Comprehensive data generation
- **Commands 14-15**: Optional additional validation

**Expected Runtime**: Commands 3-5 may take several minutes each due to Monte Carlo simulations. Individual engagement calculations (commands 6-9) run in seconds.

All results automatically save to timestamped files in the `results/` directory for documentation and repeatability.
